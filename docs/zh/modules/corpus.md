# 语料库

以侧边栏 + 编辑面板的分栏布局管理造语文本和莱比锡标注。

**导航**：点击左侧 `语料库`

用你的语言写出实际的文本，并添加学术级的行间标注。

## 文本管理

- 创建/编辑/删除语料文本
- 元数据：标题、描述、标签、时间戳
- 原始文本（造语原文）和自由翻译

## 行间标注

符合莱比锡标注规范的行间注释：

- **自动标注**：原始文本 → 按句拆分 → 按空格分词 → 从词典自动匹配
- **手动编辑**：每个 token 可独立编辑
  - 表层形式 (Surface form)
  - 语素切分 (Morpheme break)，用 `-` 分隔语素
  - 释义标签 (Gloss labels)
  - 关联词条 ID
  - IPA 音标

### 使用步骤

1. 点击 **「新建」** → 输入标题和描述
2. 在 **原始文本** 中用你的语言写一段话
3. 在 **自由翻译** 中写对应的自然语言翻译
4. 点击 **「自动标注」** — 程序尝试从词典匹配每个词
5. 对未匹配的词手动编辑

## 词典联动

token 关联词条后，悬浮显示词条详细信息（词性、释义、IPA）。

:::tip
莱比锡标注是国际语言学界通用的行间注释规范，你的语料会看起来非常专业。
:::
