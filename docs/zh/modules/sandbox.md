# 翻译沙盒

交互式翻译和标注工作台，支持两种模式，是检验你的语言是否「能用」的最佳工具。

**导航**：点击左侧 `翻译沙盒`

## 标签屈折模式

输入格式为 `gloss-TAG1-TAG2` 的文本（如 `star-PL shine-PAST in void`）：

1. 按 gloss 从词典查找对应词条
2. 按标签自动应用屈折规则
3. 生成 IPA
4. 输出跨行标注表（造语词形 / gloss 标签 / IPA）

## 句法重排模式

1. 逐个添加 token 并标注语法角色：
   - `star` → 主语 (S)
   - `shine` → 动词 (V)
   - `void` → 修饰语 (Mod)
2. 按你设置的语序自动重排
3. 自动推断屈折标签（如 S → NOM, O → ACC）
4. 应用屈折后输出跨行标注

## 词形助手

输入测试词 + 选择词性 + 选择维度值 → 实时预览屈折结果和 IPA。用于快速测试单个词的特定屈折形式。
