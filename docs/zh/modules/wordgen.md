# 造词生成器

基于音位配列规则随机生成候选词汇。

**导航**：点击左侧 `造词生成器`

有了音系规则后，就可以让程序自动生成符合规则的词了！

## 使用流程

1. 设置 **生成数量**（建议初次 50~100 个）和 **音节数范围**（建议 1~3 音节）
2. 点击 **「生成」**
3. 浏览候选词列表，每个词旁边会显示自动生成的 IPA
4. 删除不喜欢的词，为喜欢的词标注词性
5. 勾选想要保留的词
6. 点击 **「导入词典」**

## 功能参数

| 功能 | 说明 |
|------|------|
| 生成参数 | 数量（1~200）、最小/最大音节数 |
| 音节结构 | 自动读取音位配列设置，展开可选项 |
| 黑名单过滤 | 排除违反音位配列规则的生成词 |
| IPA 预览 | 每个候选词自动计算 IPA |
| 候选表 | 全选/单选、编辑词性、删除 |
| 导入词典 | 选中候选词一键导入 |

## 进阶功能

### Swadesh 100

勾选后会生成 100 个词并自动映射到 Swadesh 基本词汇表中的概念（水、火、太阳、人……），快速建立核心词汇。

### 衰减系数

滑块控制各音素类别（辅音、元音、鼻音等）的分布权重（0.0~3.0），让生成结果更符合你期望的语言风格。调高可增加某些音在词首的出现概率。

### 改写规则

使用正则表达式在生成后自动替换，适合添加长元音标记等后处理（如 `aa → ā`）。
